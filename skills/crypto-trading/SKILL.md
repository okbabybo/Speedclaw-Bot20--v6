# SKILL.md - Crypto Trading Assistant

**Skill ID:** crypto-trading
**Name:** Crypto Trading Assistant
**Description:** 专业数字资产交易辅助 - 市场分析、策略执行、风险管理

---

## 核心能力

### 📊 市场数据层
- **实时行情**：获取币安/OKX/Bybit实时价格、K线、订单簿
- **历史数据**：下载和分析历史K线数据
- **链上数据**：监控大户转账、交易所流入流出
- **情绪指标**：社交媒体情绪、恐慌贪婪指数

### 🤖 策略引擎
- **多策略并行**：同时运行多个独立策略
- **回测系统**：基于历史数据的策略验证
- **参数优化**：自动调参寻找最优参数
- **信号生成**：技术面、资金面、情绪面综合分析

### ⚠️ 风控系统
- **仓位管理**：凯利准则、固定分数法
- **止损止盈**：动态跟踪止损、分级止盈
- **风险评估**：VaR(Value at Risk)计算
- **应急预案**：极端行情自动降仓/平仓

### 📈 分析工具
- **技术分析**：MACD、RSI、布林带、斐波那契
- **量价分析**：OBV、成交量分布、资金流向
- **模式识别**：头肩顶/底、三角形、旗形等
- **指标组合**：多指标共振信号过滤

---

## 命令集合

### 查询类
- `market {symbol}` - 获取币种实时行情
- `chart {symbol} {timeframe}` - 生成K线图表
- `orderbook {symbol}` - 显示订单簿深度
- `funding {symbol}` - 显示资金费率

### 策略类  
- `strategy list` - 显示可用策略
- `strategy start {name} {params}` - 启动策略
- `strategy stop {name}` - 停止策略
- `strategy status` - 查看策略运行状态

### 交易类
- `signal {symbol}` - 获取当前交易信号
- `risk {position}` - 计算仓位风险
- `alert set {condition}` - 设置价格/指标警报
- `log {trade}` - 记录交易日志

### 报告类
- `pnl` - 显示盈亏统计
- `performance` - 策略表现分析
- `journal` - 交易日志复盘
