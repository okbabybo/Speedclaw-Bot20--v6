## 安装步骤

### 1. 安装Python 3.10+
- 访问 https://www.python.org/downloads/
- 下载并安装 Python 3.10 或更高版本
- **重要**: 安装时勾选 "Add Python to PATH"

### 2. 运行一键安装
```
双击 install.bat
```

### 3. 配置API密钥
编辑文件: `C:\chaos-lobster-trading\config\okx_config.json`
```json
{
  "apiKey": "您的OKX API Key",
  "secret": "您的OKX Secret",
  "passphrase": "您的OKX Passphrase"
}
```

### 4. 启动机器人
```
双击桌面快捷方式 "混沌龙虾交易机器人"
或运行: C:\chaos-lobster-trading\start_bot.bat
```

## 使用说明

### UI界面操作
1. **测试连接**: 点击"测试连接"按钮验证API
2. **启动机器人**: 点击"启动机器人"开始自动交易
3. **查看日志**: 右侧日志面板实时显示运行状态
4. **监控持仓**: 左侧面板显示当前持仓和盈亏

### 交易策略
- **趋势确认**: EMA9/21/50多头排列
- **MACD**: 金叉确认
- **RSI**: 35-50区间买入，50-65区间卖出
- **布林带**: 触及下轨买入，触及上轨卖出
- **支撑阻力**: 关键价位确认
- **成交量**: 放量确认

### 风控设置
- **单笔仓位**: 最大5%资金
- **止损**: 1.5倍ATR
- **止盈**: 3倍ATR (1:2盈亏比)
- **日亏损限制**: 3%资金
- **紧急停止**: 自动触发
