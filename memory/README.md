# 记忆系统架构

## 第一层：个人记忆层 (Agent独立)
- `memory/p0-longterm/` - 长期记忆，永久保留
- `memory/p1-summary/` - 重要事项与经验总结
- `memory/p2-shortterm/` - 短期闲聊记录，半月清理

## 第二层：龙虾共享记忆层
- `memory/shared/` - 重要信息、关键决策索引

## 第三层：工作空间记忆层
- `memory/workspace/` - 工作日志、自我进化记录

---

## 记录规则
1. 对话默认简体中文
2. 文档默认简体中文，代码/API保留原文
3. 输出简短，少废话
4. 只记长期有效信息，短句/列表
5. 不记完整聊天，只提炼关键
