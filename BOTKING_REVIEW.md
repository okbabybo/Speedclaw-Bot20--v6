## 三、SpeedClaw BotKing 完整规格

### 3.1 产品定位

```
名称：     SpeedClaw BotKing
类型：     现货智能交易机器人
交易对：   BTC, ETH, BNB, SOL, AVAX, XRP（可扩展）
交易所：   币安现货 USDT-M
目标用户：  有一定资金量（建议$100+）的现货交易者
核心能力： 熊市网格套利 + 牛市趋势追踪 + 全自动风控
```

### 3.2 6种市场模式

```
模式                    识别条件                              操作策略
--------------------------------------------------------------------------------
TREND_UP              EMA20(1D+4H+1H)三周期同向上        全力趋势追踪
TREND_DOWN            EMA20(4H+1H)同向下 + RSI<50         轻仓/观望
RANGE_BOUND           EMA走平 + RSI在40-60波动              全力网格套利
VOLATILE_OVERSOLD     波幅>8% + RSI<35 + 放量             网格+趋势双开
VOLATILE_OVERBOUGHT   波幅>8% + RSI>65 + 放量             全部止盈观望
CRISIS                日线RSI>80 或 <20                     全部暂停30分钟
```

### 3.3 双引擎说明

```
网格引擎（GridEngine）：
  - 用途：震荡行情 + 超卖反弹
  - 适合：RANGE_BOUND, VOLATILE_OVERSOLD
  - 原理：价格区间划分，低买高卖反复收割
  - 止盈：每格独立，+0.4%~1.0%
  - 止损：每格独立，-12%
  - 追踪：利润>6%激活，回撤3%锁定

趋势引擎（TrendEngine）：
  - 用途：单边趋势行情
  - 适合：TREND_UP, VOLATILE_OVERSOLD
  - 原理：EMA确认方向，持有直到趋势破坏
  - TP1：+15% → 止盈50%仓位
  - TP2：+25% → 止盈剩余50%
  - 止损：-12%
  - 追踪：+15%激活，回撤5%锁定
```

### 3.4 资金分配方案

```
余额分级（现货无杠杆，可大胆用）：

  $20-50    → 每币 $50   → 最多1个币（极端保守）
  $50-200   → 每币 $150  → 最多2个币
  $200-1000 → 每币 $500  → 最多3个币
  >$1000    → 每币 $1500 → 最多3个币

开仓比例：
  TREND_UP           → 每币 × 100%
  RANGE_BOUND        → 每币 × 80%
  VOLATILE_OVERSOLD  → 每币 × 120%（透支20%）
  TREND_DOWN         → 每币 × 30%（轻仓）
  VOL_OVERBOUGHT     → 每币 × 0%（禁止）
  CRISIS             → 每币 × 0%（禁止）

分批建仓（所有模式）：
  第1批：先开2格（或1格根据资金）
  第2批：止盈确认后，延迟5分钟再开下2格
  效果：避免一次性暴露全部资金
```

### 3.5 止盈止损矩阵

```
【网格引擎】
每格止盈：+0.4% ~ +1.0%（由ATR决定）
每格止损：-12%（固定）

追踪止损（网格）：
  触发：利润 > 6%
  执行：回撤3% → 全格卖出
  效果：锁定利润，不全吐回去

【趋势引擎】
第一目标：+15% → 止盈50%仓位
第二目标：+25% → 止盈剩余50%

追踪止损（趋势）：
  +20%以上 → 回撤8%触发
  +15%以上 → 回撤5%触发
  +10%以上 → 回撤3%触发

止损（趋势）：
  固定：-12%
  趋势破坏：价格 < EMA20×0.96 → 立即止损
  RSI过热：RSI > 75 → 立即止盈
```

### 3.6 8层风控矩阵

```
层级  触发条件              保护动作                    恢复条件
--------------------------------------------------------------------------------
1     余额 < $11           禁止开仓                    充值后自动恢复
2     单日亏损 > 8%        暂停1小时                   1小时后自动恢复
3     余额从高点跌20%      全部止损 + 锁定30分钟        30分钟后自动恢复
4     止损1次             冷静期5分钟                 5分钟后自动恢复
5     止损2次             冷静期10分钟                10分钟后自动恢复
6     止损3次（熔断）     暂停15分钟                  15分钟后+市场确认
7     3连亏+熊市模式      暂停至市场转势              等TREND_UP信号
8     日线RSI>80/<20      全部平仓暂停30分钟           30分钟后重新评估
```

### 3.7 手动平仓识别

```
原理：对比"状态文件记录的持仓" vs "币安API实际持仓"

执行：每5分钟检查一次

识别到手动平仓后：
  ① 标记该仓为"手动平仓"
  ② 日志：[⚠️ 手动平仓识别]
  ③ 释放资金到可用余额
  ④ 不触发追踪止损（用户主动操作）
  ⑤ 冷却期5分钟（防止重复识别）
```

### 3.8 复利与提盈

```
复利（100%）：
  每笔止盈利润 → 50%锁定 + 30%复利 + 20%buffer
  锁定部分 → 保留不动
  复利部分 → 自动投入下一格/下一仓

提盈（达到目标时）：
  触发：余额 > 历史最高 × 120%
  最低门槛：利润 ≥ $5
  提取比例：利润的 50%
  操作：记录并保留，不投入市场

举例：
  初始 $1000 → 涨到 $1240
  提盈：(1240-1000)×50% = $120
  新高点：1240×90% = $1116
  实际提取：$120  （剩余$1120继续滚）
```
