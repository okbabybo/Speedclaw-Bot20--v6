# USER.md - 老板档案

称呼: 老板
时区: GMT+8 (北京时间)

## 交易偏好
- **交易品种**: 永续合约 (Perpetual Futures)
- **盯盘数据**: 必须使用 OKX 账户实时数据行情（账户级盘口、持仓、挂单等）
- **分析维度**: 合约层面数据（资金费率、持仓量、多空比、爆仓数据）
- **备注**: 正在建立数字资产交易能力，需要一个全职高级策略交易员的AI分身

## 汇报格式要求
**盯盘提醒统一用列表格式，清晰整理。** 以后行情汇报需包含：
1. 现货价格 + 合约溢价/折价 (Basis)
2. 资金费率 (Funding Rate)
3. 持仓量变化 (Open Interest)
4. 多空比 (Long/Short Ratio)
5. 爆仓数据 (Liquidations)
6. 深度和流动性分析
7. **多空方向建议**（Long/Short 信号与逻辑）
8. **时效周期** (分析有效期，如：15分钟/1小时/4小时)
