## 二、完整资金分配方案

### 2.1 仓位上限（风控红线）

```
总仓位上限 = 余额 × 150%（可透支30%做网格）
单币仓位 = 余额 ÷ 活跃币种数 × 风险系数
风险系数：TREND_UP=1.0 | RANGE_BOUND=0.8 | TREND_DOWN=0.3 | 其他=0.5
```

### 2.2 下单金额计算

```
基础单量 = 余额 × 10% ÷ 当前价格

熊市系数 = 0.5（资金减半）
网格系数 = 1.0（每格等额）

实际单量 = 基础单量 × 熊市系数 × 网格系数

最小下单 = $11（币安现货门槛）
最大单币 = 余额 × 30%
```

### 2.3 动态调整规则

```
1. 余额 > $1000  → 每币最高$300
2. 余额 $200-1000 → 每币最高$150  
3. 余额 $50-200  → 每币最高$50
4. 余额 $20-50   → 每币最高$20（只能开1个币）
5. 余额 < $20    → 暂停，等待充值
```

## 三、止盈止损体系（核心！）

### 3.1 网格引擎止损止盈

```
每格止损 = 买入价 × (-12%)
每格止盈 = 买入价 × (+0.6%)
追踪止损 = 利润 > 6%后激活，回撤3%触发

网格平仓规则：
- 正常市：价格到达目标位 → 自动止盈
- 止损线：跌破-12% → 全部止损
- 追踪线：利润>6%后从高点回撤3% → 锁定利润
- 手动识别：用户平仓 → 同步状态，释放资金
```

### 3.2 趋势引擎止损止盈

```
目标止盈 = 买入价 × (+25%)
止损线   = 买入价 × (-12%)

追踪止损（从峰值回撤）：
- 利润 > 20% → 回撤8%触发
- 利润 > 15% → 回撤5%触发
- 利润 > 10% → 回撤3%触发
- 利润 > 5%  → 回撤1.5%触发

趋势破坏止损：
- 价格跌破EMA20 × 0.96 → 立即止损
- RSI > 75（过热）→ 立即止盈
- 4H出现死叉 → 止盈50%仓位
```

### 3.3 全局风控

```
回撤保护 = 余额从最高点跌20% → 全部止损，锁仓30分钟
单日最大亏损 = 余额 × 8% → 暂停1小时
连续亏损3次 → 暂停15分钟（熔断）
```

## 四、6种模式详细操作手册

### 模式1：TREND_UP（上升趋势）

**识别：**
- EMA20(1H)向上 + EMA20(4H)向上 + EMA20(1D)向上
- RSI(1H) > 50
- MACD histogram > 0

**操作：**
```
1. 开仓：等价格回踩EMA20附近买入
2. 仓位：每币 = 余额÷3 × 1.0（满仓）
3. 止损：入场价 - 8%
4. 目标：
   - 第一目标：+15% 止盈50%
   - 第二目标：+25% 止盈剩余50%
5. 追踪：利润>15%激活TS，回撤5%触发
6. 网格：仅开2格（顺势持有，不频繁交易）
```

### 模式2：TREND_DOWN（下跌趋势）

**识别：**
- EMA20(1H)向下 + EMA20(4H)向下
- RSI(1H) < 50

**操作：**
```
1. 开仓：禁止！不逆势抄底
2. 已有仓位：
   - 网格仓位：跌到下轨全部止损
   - 趋势仓位：立即止损
3. 空仓：等待，不开新仓
4. 观察：RSI<25时开始关注，等待止跌信号
```

### 模式3：RANGE_BOUND（震荡行情）

**识别：**
- EMA20走平（方向不明）
- RSI(1H) 在 40-60 之间波动
- 布林带收窄（BB宽度<5%）

**操作：**
```
1. 开仓：价格触碰布林带下轨买入
2. 仓位：每币 = 余额÷3 × 0.8
3. 网格：6格，每格间隔 = (上轨-下轨)÷6
4. 每格止盈：+0.6%
5. 每格止损：-12%
6. 追踪：利润>6%激活，回撤3%触发
7. 资金循环：止盈后资金自动分配到下一格
```

### 模式4：VOLATILE_OVERSOLD（高波动超卖）

**识别：**
- 高波动（日内波幅>8%）
- RSI(1H) < 35
- 成交量放大（>1.3倍均量）
- 可能出现V形反弹

**操作：**
```
1. 开仓：RSI<35 + BB下轨 + 放量 → 立即买入
2. 仓位：每币 = 余额÷3 × 1.2（透支20%）
3. 网格：4格（快速波动需要更宽格距）
4. 每格止盈：+1.0%（波动大，格距宽）
5. 止损：-10%
6. 追踪：利润>8%激活，回撤4%触发
7. 趋势确认：若价格反弹站上EMA20 → 转趋势引擎
```

### 模式5：VOLATILE_OVERBOUGHT（高波动超买）

**识别：**
- 高波动（日内波幅>8%）
- RSI(1H) > 65
- 成交量放大 + 价格快速上涨

**操作：**
```
1. 已有仓位：全部止盈！
2. 空仓：禁止开新仓
3. 观察：等RSI回到50以下再操作
```

### 模式6：CRISIS（极端行情）

**识别：**
- 日线RSI > 80（极度超买）
- 或 日线RSI < 20（极度超卖）
- 波动率异常放大

**操作：**
```
1. 全部止损/止盈
2. 所有引擎暂停
3. 等待30分钟，重新评估
4. 恢复正常后重新判断模式
```

## 五、手动干涉平仓识别

### 识别逻辑

```
每5分钟检查一次持仓状态：
1. 对比状态文件记录的持仓 vs 币安API返回的实际持仓
2. 如果状态文件有持仓，但API没有 → 用户手动平仓
3. 立即：
   - 标记该仓为"手动平仓"
   - 不执行止损/止盈追踪
   - 释放该仓资金到可用余额
   - 记录到日志：[⚠️ 手动平仓识别]
4. 不重复识别（冷却期5分钟）
```

```
┌─────────────────────────────────────────────┐
│              初始本金 = $100                 │
└─────────────────┬───────────────────────────┘
                  ↓
          每笔利润100%复利
                  ↓
    ┌─────────────┼─────────────┐
    ↓             ↓             ↓
  网格利润    趋势利润     提盈
 100%再投    100%再投    满20%提
    ↓             ↓
  开新仓      开新仓
    ↓             ↓
         滚雪球
```

**提盈规则：**
```
触发条件：余额 > 历史最高 × (1 + 20%)
最低提盈：$5
提盈比例：利润的50%
提盈后：新高点 = 当前余额 × 0.9（保留90%继续滚）
```
