## 四、持仓第一天：触发止盈

### 场景A：SOLUSDT 先涨后跌（典型震荡）

```
时间     价格       事件                    盈亏
---------------------------------------------------------------
10:00   $69.20    开仓买入格0              投入$75
10:15   $69.65    触及TP → 格0止盈        +$0.45（+0.6%）
10:15   $69.65    格1自动买入             投入$75
10:45   $69.30    格1触及TP → 止盈       +$0.42（+0.56%）
10:45   $69.30    格2自动买入             投入$75
11:30   $68.80    下跌，格2未触发行情     持有中
12:00   $68.20    继续跌至格3            买入格3
12:45   $67.00    触及SL → 全部止损      -$9.00（-12%）
```

**第一天 SOL 总结：**
```
总交易次数：4笔（2止盈 + 1加仓 + 1止损）
总盈亏：+$0.87 + $0.42 - $9.00 = -$7.71
胜率：2/4 = 50%
```

> ⚠️ **发现问题2：** 震荡行情里，格0和格1止盈了，但格2和格3继续持有时遭遇止损。
> 网格设计缺陷：**没有对已止盈格子的资金做止损保护**。
> 应该在第2次止盈后，把利润锁定，不全数投入下一格。

### 场景B：BTCUSD 持续下跌（熊市）

```
时间     价格       事件                    盈亏
---------------------------------------------------------------
10:00   $62,500   开仓格0                投入$75
10:30   $61,800   继续下跌格1买入         投入$75
11:00   $61,100   格2买入               投入$75
11:30   $60,400   格3买入               投入$75
12:00   $59,700   继续跌，未触发行情    4格全持有中
12:30   $55,000   触及SL → 4格全部止损  -$36.00（-12%）
```

**BTC 止损时实际亏损：**
```
每格止损：$75 × 12% = $9 × 4格 = $36
总账户亏损：$36 / $10,000 = -0.36%
```

> ⚠️ **发现问题3：** 4格同时持有，全部止损时亏损$36。
> 如果同时3个币都这样止损：$36 × 3 = $108 = -1.08%
> **回撤保护线：$10,000 × 0.8 = $8,000 → 不触发**
> 但单日连续亏损可能触发日亏8%保护。

## 六、修复后的改进方案

### 改进1：分批建仓（先2后2）

```
原方案：4格同时建仓，风险集中
新方案：
  第1批：先开2格（格0、格1）
  触发止盈 → 利润50%锁定
  第2批：再开下2格（格2、格3）

效果：
  - 如果行情反弹，格0、1止盈，锁住利润
  - 只有盈利后才开格2、3，避免一次性暴露全部资金
  - 亏损时只有2格在场，损失减半
```

### 改进2：止盈利润分层

```
止盈后的利润分配：
  50% → 锁定，不投入（提取或保留）
  30% → 投入下一格
  20% → 作为安全垫buffer

示例：
  格0止盈赚$0.45
  → $0.23锁定
  → $0.13投入格1（加仓）
  → $0.09作为buffer
```

### 改进3：熊市只开2格

```
熊市（TREND_DOWN）：
  - 只开2格（格0、格1）
  - 每格资金加倍：$150/格
  - 止损范围扩大：±8%（ATR自适应）

效果：
  - 资金不过度暴露
  - 如果继续跌，2格止损 = $150×2×12% = $36
  - 比4格全开少亏一半
```

### 改进4：止损后冷静期

```
止损触发后：
  冷静期 = max(5分钟, 到下一个市场扫描周期)
  冷静期内：
    - 不开新仓
    - 不判断BUY信号
    - 等待下一个3分钟扫描确认方向
```

## 十、核心改进清单

| 改进 | 原逻辑 | 新逻辑 |
|------|--------|--------|
| 资金利用率 | 9% ($300×3/$10k) | 12-15% (提升到$500/币) |
| 建仓方式 | 4格同时开 | 先开2格，止盈后再开2格 |
| 利润锁定 | 100%复利 | 50%锁定 + 30%复利 + 20%buffer |
| 熊市策略 | 轻仓0.3倍 | 只开2格+观望，等止跌信号 |
| 止损反手 | 无等待 | 冷静期5分钟 |
| 网格数量 | 固定6格 | ATR自适应3-6格 |
| 同币双引擎 | 可同时开 | 互斥，一个币只能一种策略 |
